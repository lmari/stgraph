<h2>STGraph - Boolean operators</h2>
Each of the following operators deals with with the true and false boolean values, which are defined in STGraph as follows:
- each value greater than zero (actually: greater or equal to <code>EPSILON</code>) is evaluated as true;
- each value less or equal than zero (actually: less to <code>EPSILON</code>) is evaluated as false.
The only primitive type in STGraph is double: hence, true and false are implemented as <code>1.0</code> and <code>0.0</code> respectively.
