<h2>STGraph - size</h2>
<u>Format</u>: <code>size(x)</code>
<u>Constraints</u>: <code>x</code> is an array
<u>Description</u>: size of <code>x</code>

Equivalent to the operator <code>@x</code>. The value is zero if <code>x</code> is a scalar, and a vector otherwise.
<br><code>size</code> is <a href="functions_en.properties">an array function</a>

<u>Exceptions</u>: no
